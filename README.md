# Address Book

> Django address book project, providing users with a seamless and organized solution for managing contacts. This application simplifies the process of storing, updating, and accessing contact information, enhancing efficiency in personal or professional networking.

## 💻 Requirements

Before you start, make sure you meet the following requirements:

- Have you installed the latest version of `Python`
- You have a machine `<Windows / Linux / Mac>`.

## 🚀 Installing Address Book

To install Address Book, follow these steps:

Linux and macOS:

```
1 - git clone https://github.com/fellipemarconi/Address-Book.git
2 - python3 pip install -r requiments.txt
3 - python3 -m venv venv
```

Windows:

```
1 - git clone https://github.com/fellipemarconi/Address-Book.git
2 - python3 pip install -r requiments.txt
3 - python3 -m venv venv
```

## ☕ Use Address Book

To use Address Book, follow these steps:

Linux and macOS:

```
1 - . venv/bin/activate
2 - python manage.py runserver (If you'll use Docker, please read the documentation)
```

Windows:

```
1 - .\env\Scripts\activate
2 - python manage.py runserver (If you'll use Docker, please read the documentation)
```
